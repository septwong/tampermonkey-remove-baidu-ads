# 百度/必应搜索框净化助手

[English](./README-en.md)

Tampermonkey 脚本，净化百度和必应搜索页面，去除搜索框中的热门推荐内容。

## 功能

1. 去除百度搜索框自动加载的热搜内容词条，清空 placeholder
2. 去除百度首页的 "即刻体验百度AI搜" 引导元素
3. 去除必应搜索框的热门推荐内容

## 安装

### 方法一：GreasyFork 安装（推荐）

访问 [GreasyFork 页面](https://greasyfork.org/scripts/522389) 直接安装。

### 方法二：手动安装

1. 安装 Tampermonkey 浏览器扩展
2. 创建新脚本，复制 [UserScript.js](UserScript.js) 的内容
3. 保存即可

## 支持的搜索引擎

- 百度 (www.baidu.com)
- 必应 (www.bing.com)
