# Baidu/Bing Search Box Cleaner

[中文](./README.md)

Tampermonkey script that cleans up Baidu and Bing search pages by removing hot search recommendations from search boxes.

## Features

1. Removes hot search keywords from Baidu search box and clears placeholder
2. Removes "即刻体验百度AI搜" guide element from Baidu homepage
3. Removes hot search recommendations from Bing search box

## Installation

### Method 1: GreasyFork (Recommended)

Visit [GreasyFork page](https://greasyfork.org/scripts/522389) to install directly.

### Method 2: Manual Install

1. Install Tampermonkey browser extension
2. Create a new script and copy the content from [UserScript.js](UserScript.js)
3. Save

## Supported Search Engines

- Baidu (www.baidu.com)
- Bing (www.bing.com)
